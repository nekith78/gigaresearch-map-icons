# gigaresearch-map-icons

Route-marker icons for **GigaResearch** custom Google My Maps (`maps export --icon-style chip`).

Each icon is a small PNG combining a **category colour + glyph + visiting-order number** (variant C: white
body, coloured border, coloured number). They are referenced by URL from exported KML so My Maps can render
them; served free via jsDelivr CDN. Generated by `scripts/build_map_icons.py` in the main repo — **do not
hand-edit**, regenerate.

```
icons/<category>/<n>.png        e.g. icons/food/1.png
```

Categories: food, cafe, sight, hotel, transport, shopping, bar, nature, other. Numbers 1..40.

CDN: `https://cdn.jsdelivr.net/gh/nekith78/gigaresearch-map-icons@master/icons/<category>/<n>.png`

Contains only generated marker graphics — no code, no secrets.
